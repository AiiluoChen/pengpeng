# 砰砰 (Peng Peng) - 技术支持与隐私说明

欢迎来到“砰砰”App 的技术支持页面。本仓库包含了“砰砰”在 App Store 上线所需的技术支持网站及相关法律文档。

## 关于“砰砰”

“砰砰”是一款专为 iOS 用户设计的健康监测工具，专注于 **HRV（心率变异性）** 的实时监测与深度分析。

### 核心功能
- **心跳记录**：通过 iPhone 摄像头或 Apple Watch 捕捉每一次细微的心跳。
- **HRV 分析**：提供专业的数据解读，帮助您了解压力水平与自主神经系统状态。
- **隐私保护**：所有数据均存储在您的本地设备和个人 iCloud 中，我们承诺不收集任何个人健康数据。
- **HealthKit 集成**：完美同步 Apple 健康应用，确保数据的一致性与安全性。
